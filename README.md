# ino launcher #
unofficial minecraft java launcher. minimal, offline-capable, and stable.

> 1.12.2 only — offline & online accounts supported

---

## install

since i cant afford a real certificate (they're like $70/yr lol), you gotta trust mine manually once. sorry about that!

**1. trust the cert first**
- download `1ugun.cer`
- double click → install certificate → local machine
- place in **trusted root certification authorities**
- click through until done

**2. install the app**
- download the `.msix` and double click it
- hit install, you're done

you only have to do the cert thing once. future updates just work after that.

---

real code signing certs cost money. i used a self-signed one as a workaround.
